# JavaScript Fundamentals Reference Project

This repository contains a collection of small JavaScript examples covering **core JavaScript fundamentals**.

The project is structured as a series of simple lessons that demonstrate basic language concepts such as variables, operators, objects, loops, functions, and modules.

The goal of this repository is to serve as a **quick reference and learning resource** for JavaScript basics, especially useful when revisiting core concepts before working with frameworks, testing tools, or automation projects.

I also use it as a **foundation reference for projects involving Cypress, automation, and modern JavaScript development**.

More notes and documentation are available on my site:  
👉 https://www.radeksto.com/post/js-refrence-guide

---

# What This Project Covers

The lessons in this repository demonstrate important JavaScript fundamentals including:

- Printing output using `console.log`
- Variables (`var`, `let`, `const`)
- Primitive data types
- String concatenation and template literals
- Objects and arrays
- Comparison and logical operators
- Conditional statements (`if`, `else if`, `else`)
- Loops (`for`, `for...of`)
- Functions
- Arrow functions
- Importing and exporting modules
- Basic class usage
